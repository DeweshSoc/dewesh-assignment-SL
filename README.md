# SL Assignment - Frontend

- Made in Next.js v14 using TypeScript.
- Deployed on vercel.
- Hosted link -> https://dewesh-assignment-sl.vercel.app/


## Local setup
- clone the repo
- run `cd dewesh-assignment-SL`
- run `npm install .`
- run `npm run dev` 
- .env file content ->
```
NEXT_PUBLIC_BACKEND_URL=<hosted BE url>
 ```
